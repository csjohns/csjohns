Civic Participation and Democratic Legitimacy code

Here you can find full R code for imputation, model fitting and simulations, and construction of figures for my forthcomingpaper in Political Studies. Simulations and plotting are done using Professor Chris Adolph's simcf and tile packages, available from his website.

The data is available upon registration and download from the UK Data Archive here.

  • Amelia imputation and selection
  • Model fitting, simulation, and all figures. I use visualization as a core part of interpreting model results throughout the process of analysis, so code for figures is embedded in simulation and model-fitting code.
  • Model selection code for the online appendix
